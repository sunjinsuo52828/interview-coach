# Interview Coach - Roadmap 和迭代计划

## 1. 产品愿景与里程碑

### 1.1 产品愿景

> **打造最专业的技术面试模拟平台，帮助求职者通过AI模拟真实面试场景，系统提升面试能力。**

### 1.2 核心价值主张

| 价值点 | 描述 |
|--------|------|
| **个性化** | 基于真实简历+JD的针对性模拟 |
| **真实性** | 多级别面试官角色，追问机制模拟真实压力 |

| **数据驱动** | 追踪弱项，量化进步 |

### 2.3 技术实现

```python
# 单Agent架构

class InterviewCoachMVP:
    """MVP版本 - 单Agent"""

    def __init__(self):
        self.agent = InterviewAgent()
        self.state = SessionState()

    async def run(self):
        # 1. 解析
        resume = self.agent.parse_resume(resume_text)
        jd = self.agent.parse_jd(jd_text)
        gap = self.agent.gap_analysis(resume, jd)

        # 2. 配置
        config = self.get_user_config()

        # 3. 面试
        greeting = self.agent.start_interview(config)
        while not self.is_ended:
            user_input = await self.get_user_input()
            response = self.agent.chat(user_input)
            self.show_response(response)

        # 4. 评估
        report = self.agent.end_interview()
        self.show_report(report)
```

### 4.3 ReAct设计

```
用户: "HashMap底层是数组+链表"

Agent思考过程 (THOUGHT):
  "用户回答了HashMap底层数据结构，提到了数组+链表，
   但没有提到JDK8引入的红黑树。这是一个关键知识点，
   需要追问确认用户是否了解。"

Agent决策 (ACTION):
  ask_follow_up("JDK8之后HashMap有什么变化？为什么引入红黑树？")

用户回应...

Agent观察 (OBSERVATION):
  "用户知道红黑树，但说不清楚引入原因。
   这个知识点掌握程度为partial，已追问1层。"

Agent下一步决策:
  "可以再追问一层红黑树的作用，或者换下一个话题。"
```

### 4.4 反思机制设计

```python
class ReflectiveInterviewer:
    """带反思机制的面试官"""

    def end_interview(self) -> InterviewReport:
        # 生成报告
        report = self.generate_report()

        # 反思本次面试
        reflection = self.reflect_on_interview()

        # 保存反思，供下次改进
        self.save_reflection(reflection)

        return report

    def reflect_on_interview(self) -> Reflection:
        """自我反思"""
        prompt = f"""
请对本次面试进行反思：

1. 哪些问题问得好？为什么？
2. 哪些问题可以改进？如何改进？
3. 是否有遗漏的知识点？
4. 追问深度是否合适？
5. 节奏控制如何？
6. 如果再面一次，会怎么调整？

面试记录：
{self.state.conversation_history}
"""
        response = self.claude.messages.create(
            model="claude-sonnet-4-20250514",
            max_tokens=1000,
            messages=[{"role": "user", "content": prompt}]
        )

        return Reflection.from_text(response.content[0].text)
```
